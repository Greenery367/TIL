### 코드 블록 사용하기

#### Python
```python
print('hello, World!')
```
#### Java
```java
System.out.println('hello, world!');
```

- 파이썬은 `print` 함수를 사용해 텍스트를 출력한다.

#### 그런데, 왜 코드 블럭을 많이 사용할까?
- 코드의 가독성을 높이기 위해!
- 프로그래밍 언어마다 다른 가독성으로 표시해준다. 
 ```python
print("배부르당")
```
